1. Program Description

Program name: GroupLock — AI Group Scheduling

GroupLock enables event organizers to coordinate group availability by sending SMS messages to attendees. An AI assistant conducts a short conversational exchange to collect and confirm each attendee's available dates and times. Attendees receive a confirmation message when a time is finalized, plus optional reminders set by the organizer.

2. Message Frequency

Message frequency varies depending on the event. Attendees can expect to receive:

  • 1 initial availability request message
  • 1–3 follow-up messages to confirm the parsed availability
  • 1 confirmation message when the event time is finalized
  • 0–5 reminder messages at intervals set by the organizer (e.g. 1 week, 1 day, or 1 hour before the event)

For recurring events, this cycle may repeat before each occurrence.
